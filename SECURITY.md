# Security Policy

## Scope

This repository is the public MoneyClaw repo.

It contains public docs, the public skill bundle, and examples. It does not contain full internal infrastructure or production operations material.

## Reporting A Security Issue

Please do not open public GitHub issues for security-sensitive reports.

Send reports to:

- `support@moneyclaw.ai`
- `privacy@moneyclaw.ai`

Include:

- a clear description of the issue
- impact and affected surface
- reproduction steps or proof of concept
- whether you believe the issue affects the public repo, the hosted product, or both

## Safe Harbor

Please avoid:

- accessing data that does not belong to you
- degrading service availability
- social engineering
- public disclosure before remediation

## What Belongs In Public Issues Instead

Use public issues for:

- documentation problems
- missing examples
- unclear API behavior in published docs
- skill wording or UX feedback
