# FAQ

## Is this the full MoneyClaw product repo?

No. This is the public repo.

## Why is the repo curated?

Because MoneyClaw has internal infrastructure, incident history, production runbooks, and security-hardening material that should not be published alongside public docs.

## Can an agent really make payments with MoneyClaw?

Yes. The public docs and skill are intentionally honest about that. The safety boundary is about user authorization and prepaid limits, not pretending the product is read-only.

## Does MoneyClaw support pre-authorized operation?

Yes. Dashboard approval is the default, but users can deliberately enable account-level agent auto-approval when they want their connected agent to keep moving without a second dashboard click.
