# Merchant Flows

MoneyClaw also supports merchant-side payment collection.

This is a secondary public story behind the buyer-side OpenClaw checkout wedge.

## Public Merchant Capabilities

- create hosted invoices
- generate checkout links
- expose merchant settings
- support webhook-driven invoice completion

## When To Use This Path

Use merchant flows when the user explicitly wants to:

- accept payments
- create invoices
- embed checkout
- connect merchant-side automation

## Public Note

This repo only describes the public merchant surface at a high level.

It does not publish internal webhook handling, ops runbooks, or deployment/security details.
